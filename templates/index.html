API Chatbot Royal's Resto

Selamat datang di API Chatbot Royal's Resto. API ini menyediakan endpoint untuk berinteraksi dengan chatbot dan mengambil pertanyaan yang tidak dikenal.

# Endpoint Chat

URL: /chat

Method: POST

Request JSON:

{
    "pertanyaan": "Apa menu favorit di Royal's Resto?"
}

Response JSON (jawaban normal):

{
    "pertanyaan": "Apa menu favorit di Royal's Resto?",
    "jawaban": "Menu favorit kami adalah Nasi Goreng Royal.",
    "status": "ok"
}

Response JSON (ambigu):

{
    "pertanyaan": "resto",
    "opsi_pertanyaan": [
        "Apa jam buka Royal's Resto?",
        "Dimana lokasi Royal's Resto?"
    ],
    "jawaban": "Pertanyaan mana yang kamu maksud?",
    "status": "ambigu"
}

# Endpoint Pertanyaan Tidak Dikenal

URL: /pertanyaan-unknown

Method: GET

Query Parameter:

  • page (opsional, default=1) â€” nomor halaman untuk pagination

Contoh Request:

GET /pertanyaan-unknown?page=2

Response JSON:

{
    "page": 2,
    "per_page": 10,
    "total_data": 35,
    "total_pages": 4,
    "data": [
        { "id": 11, "pertanyaan": "Pertanyaan yang belum dipahami 11" },
        { "id": 12, "pertanyaan": "Pertanyaan yang belum dipahami 12" }
    ]
}

# Informasi Tambahan

  • Pastikan Content-Type pada request POST adalah application/json.
  • Jika pertanyaan tidak dikenali oleh chatbot, pertanyaan akan otomatis disimpan di database.
  • Gunakan pagination untuk mengambil semua pertanyaan yang tidak dikenal.
